sum_as_string

描述

Formats the sum of two numbers as string.

函数签名

sum_as_string(a, b) -> None

参数

此函数没有参数

返回值

示例

输入:

sum_as_string(5, 10)

输出:

'15'

输入:

sum_as_string(100, 200)

输出:

'300'

Python使用示例

import numpy as np
from rust_pyfunc import sum_as_string

# 使用示例


result = sum_as_string(5, 10)
print(f"结果: {result}")
