read_trade_fast

描述

Python 可调用：read_trade_fast(code, date, with_retreat=0, with_afternoon_adjust=False)

返回 (n, 7) float64 numpy 数组，列顺序： [time_sec, price, volume, turnover, flag, bid_order, ask_order]

time_sec 为 epoch 秒（与 prepare() 中 exchtime.astype(int64)//1e9 一致）。

与 read_trade 的差异： - flag!=32 过滤默认开启（with_retreat=0） - adjust_afternoon 可在读时一步完成（with_afternoon_adjust=True） - 不返回 DataFrame，直接返回 numpy 数组（供流水线内部使用或独立验证）

函数签名

read_trade_fast(code, date, with_retreat, with_afternoon_adjust) -> None

参数

此函数没有参数
